# Project Structure

- **frontend/**: React application (Vite)
- **backend/**: Node.js Express application

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev` (Runs on port 5000)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` (Runs on port 5173 usually)
